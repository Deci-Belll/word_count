# Clipboard Character Counter (クリップボード文字数カウンター)

クリップボードにコピーされた文字列を読み込み、瞬時に文字数をカウント・チェックするシンプルなCLI（コマンドライン）ユーティリティツールです。

SNSの投稿制限（Xなどの文字数制限）の確認や、特定の文字数以内におさめたい文章のチェックに役立ちます。

## 🛠 主な機能
- **クリップボード自動取得**: Enterキーを押すだけで、直前にコピーしたテキストを自動で読み込みます。
- **連続カウント機能**: スクリプトを実行したまま、Enterを押すだけで何度でも連続して文字数を計測できます。
- **文字数超過アラート**: あらかじめ設定した最大許容文字数（デフォルト：24文字）を超えると、`[TOO LONG!!]` という警告を表示します。

## 🚀 動作環境
- Python 3.x
- 依存ライブラリ: `pyperclip`

## 📦 準備（セットアップ）

1. **リポジトリのクローン（またはファイルのダウンロード）**
   ```bash
   git clone https://github.com/Deci-Belll/word_count.git
   cd <フォルダ名>

2. **必要な外部ライブラリのインストール**

   このツールはクリップボード操作のために `pyperclip` を使用します。ターミナルで以下のコマンドを実行してインストールしてください。
   ```bash
   pip install pyperclip

## 💻 使い方

1. **スクリプトを実行します**

   ```bash
   python word_count.py

2. **スクリプトを実行します**

   計測したい文字列をコピーします

   Webブラウザやメモ帳などで、文字数を調べたいテキストを通常通りコピー（`Ctrl + C` など）します。

3. **Enterキーを押します**

   スクリプトを実行しているターミナル上で Enterキー を押すと、コピーした文字列と文字数が表示されます。

💡 **制限文字数の変更方法**

   コード内の `MAX_LENGTH` の値を書き換えることで、アラートを出す最大文字数を自由に変更できます。
   ```python
   # 最大許容文字数（例：Xの標準ポストなら140に変更など）
   MAX_LENGTH = 140
   ```
